-Jerry To
-Capture the Honey Game
-Written using Python 3.3.3 and Pygame 
-All art assets are free clip art from www.clker.com
-Sound is from www.soundbible.com 

-Game object classes are placed in separate files. Classes include Hornet class for player's avatar, a Bee class, Larva class, and Queen class 
for enemy NPCs and separate classes for the main menu and menu options.

-To play the game, simply navigate using the direction keys on your keyboard to reach the beehive to finish the level. Avoid different enemies 
throughout the mazes in each level.
